How to run this script

  1. Go to script.google.com and click New project
  2. Delete any existing code and paste the generated script above
  3. Click the Run ▶ button (or press Ctrl+R)
  4. When prompted, click Review Permissions → choose your Google account → Allow
  5. Wait for execution to finish, then click Execution log at the bottom
  6. You'll see the Form URL — click it to open your new Google Form!
